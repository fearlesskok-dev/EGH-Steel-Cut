# SteelCut Pro - 智慧下料排版系統

專為鐵板切割設計的輕量級排版軟體，支援自動旋轉省料、DXF 匯入與切割補償設定。

## ✨ 功能特點
* **圖形化介面**：深色模式 UI，操作直觀。
* **自動排版**：內建演算法，自動計算最佳排列方式。
* **自動旋轉**：自動判斷並旋轉部件以節省材料。
* **孔位管理**：支援單孔、環形孔 (PCD) 快速設定。
* **DXF 支援**：可匯入特殊形狀 DXF，並匯出排版結果 DXF。
